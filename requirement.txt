## Requirements

This project was developed using Python and the following libraries:

* NumPy
* Pandas
* Matplotlib
* Seaborn
* SciPy
* Statsmodels
* Jupyter Notebook

### Installation

1. Clone this repository:

```bash
git clone <repository-url>
cd <repository-folder>
```

2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

### Python Version

This project was developed and tested using:

```text
Python 3.12
```

### Running the Project

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook file and run all cells to reproduce the analysis, visualizations, model fitting, assumption checks, and interpretation results.
